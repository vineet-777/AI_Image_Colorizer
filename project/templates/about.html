About This Project

VTU 5th Semester Mini Project

Project Overview

This project is developed as part of the VTU 5th Semester Mini Project requirements. The system demonstrates the implementation of "Image Colour Restoration using Deep Learning (CNN)" and provides a user-friendly dashboard to upload, process, and view restored images.

The objective of this project is to restore colours in faded or grayscale images using a trained Convolutional Neural Network model and showcase the practical application of deep learning techniques.
